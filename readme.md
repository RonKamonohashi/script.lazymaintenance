## Features
- **Cleaner Options**: Auto/Manual Soft clean trimm to folders to a specific limit, hard clean clears the folders.
- **Refresh Options**: Scans repositories for addon updates or refresh the skin if you have ui issues.
- **Log Options**: Read, export,upload or clear the Kodi log file (`kodi.log`).
- **Backup/Restore**: Creates a ZIP backup of your addons, userdata, media folder which can be restored on other devices etc.
- **Fresh Start**: Resets Kodi to its default state by deleting all addons (except Lazy Maintenance) and the userdata folder. Use with cautionâ€”all settings and customizations will be lost!


## Installation

1. **Download the Addon**:
   - Click on Code.
   - Click on Download Zip.
2. **Install in Kodi**:
   - Open Kodi and go to `Settings > Addons > Install from zip file`.
   - Navigate to the downloaded or created ZIP and select it.
   - The addon will install under `Program Addons`.
2. **Run the Addon**: Go to `Addons > Program Addons > Lazy Maintenance` to access the menu.

## Usage

- Launch the addon from the Program Addons section.
- Select an option from the menu
- Follow on-screen prompts for actions
